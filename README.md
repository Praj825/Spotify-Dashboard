# 🎵 Spotify Streaming Insights Dashboard | Power BI

## 📌 Project Overview
Analyzed **89,740+ Spotify tracks** across 114 genres and 31,430+ artists to uncover music trends.

## 🛠️ Tools Used
- **Power BI** (DAX, Power Query)
- **Dataset:** Spotify Tracks Dataset (Kaggle)

## 📊 Key Insights
| # | Insight |
|---|---|
| 1 | Average popularity score: **33.24 / 100** |
| 2 | **The Beatles & George Jones** dominate track count |
| 3 | Top songs hit popularity score of **99** |
| 4 | **91.45% tracks** are non-explicit |
| 5 | High-energy tracks tend to have higher danceability |

## 📈 Dashboard Features
- KPI cards — Total Tracks, Artists, Genres, Avg Popularity
- Top 5 artists by track count
- Top 5 most popular songs
- Energy vs Danceability scatter plot
- Explicit content distribution
